# FlexBoxChild

This component helps in constructing more complex layouts with `FlexBox`, such as mixing fluid and fixed width children.

### Usage
```tsx
import {FlexBox} from '@influxdata/clockface'
```
```tsx
<FlexBox>
  <FlexBox.Child basis="20px" grow="1" shrink="0">
    // Children
  </FlexBox.Child>
</FlexBox>
```
