# ResourceCardName

`ResourceCardName` is intended to be passed into the `name` prop in `ResourceCard`. It can be accessed via the single `ResourceCard` import as a subclass.

### Usage
```tsx
import {ResourceCard} from '@influxdata/clockface'
```
```tsx
<ResourceCard.Name />
```
