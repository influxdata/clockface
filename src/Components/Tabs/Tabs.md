# Tabs

Tabs are a great way to offer navigation within a page or component. They can be customized to suit a variety of use cases.

### Usage
```tsx
import {Tabs} from '@influxdata/clockface'
```
```tsx
<Tabs>
  <Tabs.Tab />
  <Tabs.Tab />
  <Tabs.Tab />
</Tabs>
```
