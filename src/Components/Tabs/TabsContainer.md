# TabsContainer

This component is intended to wrap `<Tabs />` and `<Tabs.TabContents />` so they line up nicely. You can also use `FlexBox` to achieve the same goal.

### Usage
```tsx
import {Tabs} from '@influxdata/clockface'
```
```tsx
<Tabs.Container>
  <Tabs>
    <Tabs.Tab />
    <Tabs.Tab />
    <Tabs.Tab />
  </Tabs>
  <Tabs.TabContent />
</Tabs.Container>
```
