# TabContents

Intended for use alongside `<Tabs />` to display the contents of the selected tab. 

### Usage
```tsx
import {Tabs} from '@influxdata/clockface'
```
```tsx
<Tabs.TabContents>
  // Contents go here
</Tabs.TabContents>
```
