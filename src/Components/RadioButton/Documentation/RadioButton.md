# Radio Button

This is a variation of an input with type radio.

### Usage

```tsx
import {Button} from '@influxdata/clockface'
```

```tsx
<RadioButton
  value="test-1"
  id="test-1"
  type={InputToggleType.Radio}
  toolTipText="tooltip text"
  size={ComponentSize.small}
  ref={radioButtonRef}
  labelText={'Radio Button'}
  onChange={() => {}}
></RadioButton>
```
