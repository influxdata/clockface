# Panel Body

Panel Body is for containing the main content of a panel. It can be accessed via the single `Panel` import as a subclass.

### Usage
```jsx
import {Panel} from '@influxdata/clockface'
```
```jsx
<Panel.Body>
  // Children
</Panel.Body>
```
