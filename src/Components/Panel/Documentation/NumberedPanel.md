# NumberedPanel

This is a composed variant of `Panel` in which a number (or step) indicator appears in the top left corner and the padding is slightly different.

### Usage
```tsx
import {NumberedPanel} from '@influxdata/clockface'

<NumberedPanel>

  // Children

</NumberedPanel>
```
