# App Wrapper

This is the outermost container for Clockface apps. We strongly recommend having this component be the only child of your react root.

### Usage
```tsx
import {AppWrapper} from '@influxdata/clockface'
```
```tsx
<AppWrapper>
  <NavMenu />
  <Page />
</AppWrapper>
```
