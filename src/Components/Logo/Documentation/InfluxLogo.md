# InfluxLogo

This is a SVG based image component provided to help standardize the brand across codebases

### Usage
```tsx
import {InfluxLogo} from '@influxdata/clockface'
```
```tsx
<InfluxLogo 
    logoMark={'Kubo'} 
    baseText={'InfluxData'} 
    auxiliaryText={'None'} 
    symbol={'Registered'}
    fill={InfluxColors.White} 
    centeredLogo={false}
/>
```
