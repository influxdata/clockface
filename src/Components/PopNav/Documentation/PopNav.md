# Pop Nav

A Popover based navigation menu.

### Usage

```tsx
import {PopNav} from '@influxdata/clockface'
```

```tsx
<PopNav>
  // Some content
  <PopNav.Item>item 1</PopNav.Item>
  <PopNav.Item>item 2</PopNav.Item>
  <PopNav.Item>item 3</PopNav.Item>
</PopNav>
```
